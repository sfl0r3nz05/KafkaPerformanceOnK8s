# Helm chart installation

```console
wget https://get.helm.sh/helm-v3.9.4-linux-amd64.tar.gz
```

```console
tar xvf helm-v3.9.4-linux-amd64.tar.gz
```

```console
sudo mv linux-amd64/helm /usr/local/bin
```

```console
rm helm-v3.9.4-linux-amd64.tar.gz
```

```console
rm -rf linux-amd64
```

```console
helm version
```
